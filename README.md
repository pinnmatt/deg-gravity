## What is DEG?

Discrete Emergent Gravity (DEG) is a theoretical physics framework proposing that spacetime consists of *N* Planck-scale discrete atoms, each governed by a single Hamiltonian constraint:

```
H = Σ_n [ p_n² / 2m  −  (α/2) a_n² ] = 0
```

Each atom carries *g* = 442 internal quantum states, fixed by the requirement that SU(21)×U(1) embeds the Standard Model gauge group. No parameter is fitted to observation.

DEG is a theory of the **post-condensation universe** — describing cosmic history from spacetime atom formation (~T_reheat ~ 10¹⁸ GeV) through today, approximately 97% of cosmic history in log energy scale.
