## What is DEG?

Discrete Emergent Gravity (DEG) is a theoretical physics framework proposing that spacetime consists of *N* Planck-scale discrete atoms, each governed by a single Hamiltonian constraint:

```
H = Σ_n [ p_n² / 2m  −  (α/2) a_n² ] = 0
```

Each atom carries *g* = 442 internal quantum states, fixed by the requirement that SU(21)×U(1) embeds the Standard Model gauge group. No parameter is fitted to observation.

## Falsifiable predictions

The programme makes 16 explicit falsifiable predictions with named experiments and timelines, including:

- *w* = −1 exactly — any 5σ detection of *w* ≠ −1 by DESI or Euclid falsifies the dark energy sector
- Normal neutrino ordering (JUNO, 3–5 yr)
- Proton decay τ_p ~ 10³³⁻³⁴ yr (Hyper-Kamiokande, 5–10 yr)
- Triple Higgs coupling κ_λ = −1.449 ± 0.17 (HL-LHC ~2035)
- Non-Gaussianity f_NL = 7 ± 2 (CMB-S4 ~2032)

## Known open problems

DEG explicitly states its open problems:

- Higgs coupling κ_V = 0.432, currently 14.2σ from LHC; resolution pathway identified
- Λ carries a factor-2 systematic from non-perturbative matching
- Baryon asymmetry η_B compatible but not precisely derived
- Galaxy cluster profiles at large *r* not yet quantitatively derived
